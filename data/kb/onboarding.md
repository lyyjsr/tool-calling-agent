# 新人接手项目指引

## 目标
帮助新成员在第一天完成项目拉取、本地运行、接口测试和基本排查。

## 基本步骤
1. 克隆代码仓库并切换到主开发分支。
2. 创建并激活本地 Python 虚拟环境。
3. 安装 requirements.txt 中的依赖。
4. 准备本地 .env 文件，填写必要配置项。
5. 使用 uvicorn 启动服务。
6. 访问 /health 和 /docs 确认服务状态。

## 本地运行建议
- 优先确保 Python 版本与项目要求一致。
- 如果接口启动失败，优先检查依赖安装和环境变量。
- 如果数据库初始化失败，先确认 SQLite 文件权限和路径是否正确。

## 提交代码前检查
- 运行 pytest，确保测试通过。
- 确认 .env 没有被加入 Git。
- 确认数据库文件没有被提交到仓库。
